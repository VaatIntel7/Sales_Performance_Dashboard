# Sales Performance Dashboard

A clean, interactive **Sales Performance Dashboard** built in Microsoft Excel to track revenue, profit, customers, and sales performance across products, cities, and sales reps.

## Key Metrics

- **Total Customers**: 2,098  
- **Total Profit**: ₦465,674,000  
- **Total COGS**: ₦1,862,696,000  
- **Total Revenue**: ₦2,328,370,000  

## Insight Summary

- **Peter** is the top-performing sales representative, generating ₦434,805,000 in revenue.
- **Sofa Classic** and **Laptop A13** are the most profitable products.
- **Lagos** has the highest Cost of Goods Sold (COGS), indicating strongest sales volume.
- Customer numbers remained stable from January to April, with a sharp drop in May.
- **April** recorded the highest number of customers (536).

## Analytical Dimensions 

- Profit analysis by product  
- Revenue by Sales Representative  
- Monthly customer trends  
- COGS breakdown by city  
- Interactive slicers for Region, Category & Product  

## How I Built This 

1. I organized sales data in `Sales Data.xlsx`
2. Created multiple PivotTables to summarize metrics (profit, revenue, COGS, customers)
3. Built interactive Pivot Charts for visual analysis
4. Designed a professional dashboard layout with slicers for easy filtering
5. Added key KPIs and arranged charts for clear storytelling

No macros or VBA were used — everything is built with native Excel features (PivotTables, Slicers, and Charts).
